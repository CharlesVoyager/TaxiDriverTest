# TaxiDriverTest

計程車執業登記證模擬考(電腦版)

一款以 C# / WPF 開發的桌面應用程式,協助使用者練習「計程車駕駛人執業登記證」考試題目,涵蓋交通法令與各縣市地理環境題庫。

## 功能特色

- **多題庫切換**:目前內建以下考科,點擊按鈕即可切換:
  - 交通法令
  - 臺北市地理環境
  - 新北市地理環境
  - 基隆市地理環境
  - 桃園市地理環境
  - 宜蘭縣地理環境
- **隨機出題**:每次測驗從題庫中隨機抽取是非題與選擇題各半,共 20 題。
- **即時作答與導覽**:畫面提供題號按鈕,可任意跳題作答與檢視。
- **對錯批改**:交卷後在每一題號旁以 V / X 標示作答正確與否。
- **題庫可擴充**:題目以純文字 Markdown 檔案儲存於 `QuestionBank/` 資料夾,方便直接編輯或新增題目而不需修改程式碼。

## 題庫格式

題庫檔案為純文字 Markdown,程式會依固定格式解析,新增或修改題目時請維持相同格式。

**選擇題**(每題 3 個選項,正確答案以 `**` 包住):

```
1. 題目文字
選項一
**正確選項**
選項三
```

**是非題**(答案以「是」或「否」表示):

```
1. 題目文字
**是**
```

> 選擇題與是非題分別存放在不同檔案中(檔名結尾為「選擇題」與「是非題」)。

## 執行環境需求

- Windows 作業系統
- [.NET 10 SDK](https://dotnet.microsoft.com/)(專案目標框架為 `net10.0-windows`,並使用 WPF)

## 建置與執行

```bash
git clone https://github.com/CharlesVoyager/TaxiDriverTest.git
cd TaxiDriverTest
dotnet build
dotnet run
```

或直接使用 Visual Studio 開啟 `TaxiDriverTest.slnx` 並執行。

## 使用方式

1. 啟動程式後預設載入「交通法令」題庫,開始 20 題隨機測驗。
2. 點擊上方按鈕可切換至其他縣市地理環境題庫,並重新產生一組隨機題目。
3. 依序作答,亦可透過題號按鈕跳至任一題檢視或修改答案。
4. 完成後點擊交卷按鈕,程式會在每題號旁標示 V(答對)或 X(答錯)。
